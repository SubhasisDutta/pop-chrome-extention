## Getting Started

### Installation

1. **Load the Extension**
   - Open Chrome and go to `chrome://extensions/`
   - Enable "Developer mode" in the top right corner
   - Click "Load unpacked" and select the POP folder
   - You'll see the POP icon (purple "P") in your toolbar

2. **First Launch**
   - Click the POP icon or press `Alt+P` to open the dashboard
   - Sample data is pre-loaded so you can explore all features
   - The dashboard opens in a new tab as a full-page application

3. **Quick Access**
   - Click the toolbar icon for a quick-action popup
   - Use keyboard shortcuts for instant access to any bubble
   - Right-click on selected text to capture it directly

## Dashboard Overview

### Layout

The dashboard uses a **Bento Grid** layout - a modern, organized arrangement of cards that automatically adjusts to your screen size. The UI features a beautiful glassmorphism design with sakura pink accents.

### Two View Modes

1. **Grid View** (Default): All enabled bubbles displayed in a bento grid layout
2. **Expanded View**: Single bubble in full-screen with sidebar navigation
   - Click the expand button (â›¶) in any bubble header to enter expanded view
   - A sidebar appears on the left with icons for all enabled bubbles

   - Click any sidebar icon to switch between bubbles in expanded view

### Interactive Tooltips

Each bubble has an info icon (?) next to its title. Hover over this icon to see a helpful tooltip explaining what the bubble does and how to use it.

### Theme Toggle

- Click the sun/moon toggle in the header to switch between light and dark mode

- **Settings (gear icon)**: Enable/disable bubbles, configure schedules

- **Theme Toggle**: Switch between light and dark mode

## Bubble Guide

### 1. Cognitive Offload

**Purpose**: Capture thoughts instantly without breaking your workflow

**Hotkey**: `Alt+C`

**How to Use**:

1. Press `Alt+C` anywhere in Chrome to open the capture overlay
2. Type your thought in the text field
3. Choose a tag:
   - **Actionable**: Something you need to do
   - **Reference**: Information to save for later
4. Press Enter or click a tag to save
5. The overlay disappears, keeping you in flow

**Features**:
- View all captured thoughts in the dashboard
- Mark thoughts as completed
- Filter by Actionable/Reference
- Delete unwanted items

**Best Practice**: Use this every time a thought interrupts your focus. The act of writing it down and tagging it helps clear your mind.

---

### 2. Month Cash Flow

**Purpose**: Track monthly income and expenses

**Hotkey**: `Alt+M`

**How to Use**:

1. Click "+ Add Transaction" to log income or expenses
2. Select the type (Income/Expense)
3. Enter the amount and category
4. Add an optional description
5. Set the date (defaults to today)

**Features**:
- Monthly summary showing Income, Expenses, and Balance
- Category breakdown for expenses
- Recent transaction history
- Custom categories for income and expenses

**Best Practice**: Log transactions as they happen or do a daily 5-minute review.

---

### 3. Net Worth Logger

**Purpose**: Track your total wealth over time

**Hotkey**: `Alt+N`

**How to Use**:

1. Click "+ Update Net Worth" to add a new entry
2. Enter your current values for:
   - **Assets**: Cash, Investments, Property, Other
   - **Liabilities**: Credit Card, Loans, Mortgage, Other
3. Save the entry

**Features**:
- Current net worth display with change from previous
- Visual history chart (last 6 months)
- Breakdown of assets vs liabilities

**Best Practice**: Update monthly on a consistent day (e.g., the 1st).

---

### 4. Stock Watchlist

**Purpose**: Track stocks you're interested in

**Hotkey**: `Alt+S`

**How to Use**:

1. Create a watchlist (e.g., "Tech Stocks", "Index Funds")
2. Add stocks by entering:
   - Stock symbol (e.g., AAPL)
   - Company name (optional)
   - Exchange (NASDAQ, NYSE, etc.)
3. Click quick links to view on Google Finance, Yahoo Finance, or Robinhood

**Features**:
- Multiple watchlists for organization
- Quick links to financial sites
- Remove stocks you no longer track

**Best Practice**: Create themed watchlists for different investment strategies.

---

### 5. Purpose Gatekeeper

**Purpose**: Ensure every task connects to a meaningful goal

**Hotkey**: `Alt+G`

**How to Use**:

1. Create your life purposes (e.g., "Career Growth", "Health")
2. When adding a task:
   - Write what you want to do
   - Link it to a purpose (or create a new one)
   - Mark whether you **choose** to do it or **have** to do it
3. If you can't assign a purpose, consider deleting the task

**Features**:
- Autonomy Score: Shows ratio of chosen vs. obligated tasks
- Purpose color-coding
- Challenge prompt for purposeless tasks

**Best Practice**: High Autonomy Score = more meaningful work. Aim for 50%+.

---

### 6. Daily Negotiator

**Purpose**: Plan each day with balance between hard work and rewards

**Hotkey**: `Alt+D`

**How to Use**:

1. Each morning, click "Start Planning Your Day"
2. Answer: "What does a day look like that would make you want to wake up tomorrow?"
3. Add tasks, marking each as:
   - **Regular**: Normal tasks
   - **Hard**: Challenging tasks requiring effort
   - **Reward**: Things you enjoy
4. The system prompts you to add rewards after hard tasks

**Features**:
- Balance indicator between hard tasks and rewards
- Progress tracking throughout the day
- Historical plan archive

**Best Practice**: If you have 3 hard tasks, add at least 3 rewards.

---

### 7. Question Primer

**Purpose**: Focus deep work by clarifying what you're trying to solve

**Hotkey**: `Alt+Q`

**How to Use**:

1. Before starting deep work, click "+ Set Core Question"
2. Write the ONE question you're trying to answer
3. Associate it with the site/context you'll be working in
4. Mark questions as resolved when answered

**Features**:
- Tracks stuck questions (open for 3+ days)
- Suggests reframing stuck questions
- History of resolved questions

**Best Practice**: Never start deep work without setting a core question first.

---

### 8. Flow Thermometer

**Purpose**: Stay in the optimal challenge zone

**Hotkey**: `Alt+F`

**How to Use**:

1. When prompted (or manually), rate:
   - **Task Difficulty**: 1-10 (how hard is this?)
   - **Your Skill Level**: 1-10 (how capable are you?)
2. See your current state:
   - **Flow**: Difficulty â‰ˆ Skill (optimal!)
   - **Anxiety**: Difficulty > Skill (break it down)
   - **Boredom**: Skill > Difficulty (add challenge)
3. Follow the suggestions

**Features**:
- Subtle notification every 30 minutes (configurable)
- Pauseable during meetings
- Daily flow statistics

**Best Practice**: Check in honestly. Adjusting task difficulty is the key to sustained motivation.

### 9. Truth Time Logger

**Purpose**: See the reality of how you spend time online

**Hotkey**: `Alt+T`

**How to Use**:

1. Categorize websites as "Deep" or "Shallow":
   - **Deep**: Focused, productive work (GitHub, Docs)
   - **Shallow**: Email, social media, news
2. Log time spent on each category
3. Check your Truth Score

**Features**:
- Truth Score percentage (Deep / Total time)
- Quick log buttons for 15/30/45/60 minutes
- Site categorization memory

**Best Practice**: Be brutally honest. The score doesn't judge - it just shows truth.

---

### 10. Tab Snoozer

**Purpose**: Manage tab chaos using the 2-minute rule

**Hotkey**: `Alt+B`

**How to Use**:

1. When a tab has been idle, you'll see an indicator
2. Choose from three options:
   - **Do It**: If it takes < 2 minutes, do it now
   - **Snooze It**: Close and reopen tomorrow
   - **Trash It**: Close forever
3. Snoozed tabs automatically reopen at the scheduled time

**Features**:
- Automatic idle detection (5 minutes)
- Custom snooze durations
- List of upcoming snoozed tabs

**Best Practice**: Be aggressive with Trash. If you won't miss it, you don't need it.

---

### 11. Mastery Graph

**Purpose**: Track incremental improvement in ONE metric

**Hotkey**: `Alt+Y`

**How to Use**:

1. Choose what to track (e.g., "Lines of Code", "Pages Read")
2. Log today's value
3. Compare to yesterday:
   - **Green**: Today > Yesterday (improvement!)
   - **Yellow**: Same or less (keep trying)
4. Build streaks

**Features**:
- Only shows Today vs Yesterday (no anxiety-inducing long-term view)
- Streak counter
- Customizable metric

**Best Practice**: Pick ONE meaningful metric. Consistency beats intensity.

---

### 12. Digital Cleaner

**Purpose**: Maintain a clean bookmark collection

**Hotkey**: `Alt+K`

**How to Use**:

1. Old bookmarks (6+ months) appear for review
2. For each bookmark, decide:
   - **Archive**: Move to archived folder
   - **Delete**: Remove permanently
   - **Skip**: Review later
3. Clear your digital clutter

**Features**:
- Automatic detection of old bookmarks
- Archived bookmarks are preserved but organized
- Progress tracking

**Best Practice**: Process a few bookmarks each time you open the browser.

---

### 13. Weekly Review

**Purpose**: Reflect on your week and plan the next

**Hotkey**: `Alt+W`

**How to Use**:

1. When prompted (or manually), start the review
2. Answer 5 catalytic questions:
   - What went well this week?
   - What could have gone better?
   - What is the 80/20 of next week?
   - What am I avoiding?
   - What would make next week great?
3. Save your responses

**Features**:
- Scheduled prompts (configurable day/time)
- Dismissible or locked mode
- Download reviews as text files
- Review history

**Best Practice**: Protect this time. 15 minutes of reflection saves hours of misdirection.

---

### 14. Life Expectancy Calculator

**Purpose**: Perspective on time and financial freedom

**Hotkey**: `Alt+L`

**How to Use**:

1. Enter your data:
   - Date of birth
   - Expected lifespan (default: 85)
   - Weekly usable hours
   - Current net worth
   - Monthly spending & savings
2. View calculated metrics

**Calculated Metrics**:
- Remaining weeks and hours
- Life progress percentage
- Hour worth (Net worth / Remaining hours)
- Auto Income (1% annual return / 12)
- Freedom Aim (Net worth needed for passive income to cover expenses)
- Freedom Progress percentage

**Best Practice**: Update monthly with your net worth changes.

## Settings & Configuration

### Accessing Settings

Click the **Settings** icon in the dashboard header.

### Enable/Disable Bubbles

Toggle individual bubbles on or off based on your needs. Disabled bubbles are:
- Hidden from the dashboard grid view
- Hidden from the sidebar navigation in expanded view
- Excluded from keyboard navigation

This allows you to customize POP to show only the utilities you actively use.

### Weekly Review Schedule

- **Day**: Choose which day of the week
- **Time**: Set the specific time
- **Dismissible**: Allow skipping or rescheduling

### Flow Check Settings

- **Interval**: How often to prompt (default: 30 minutes)
- **Pauseable**: Allow pausing during meetings

### Reset Data

Use with caution! This deletes all data and reinitializes with sample data.

## Tips & Best Practices

### Morning Routine
1. Open Daily Negotiator and plan your day
2. Check Weekly Review if it's Friday
3. Set your core question for the first deep work session

### During Work
1. Use Cognitive Offload whenever thoughts interrupt (Alt+C)
2. Expand bubbles to full-screen when you need focused work
3. Use sidebar navigation to quickly switch between bubbles
4. Check Flow Thermometer when prompted
5. Log time honestly in Truth Logger
6. Hover over info icons (?) if you need a reminder of what a bubble does

### Evening Review
1. Mark completed tasks in Daily Negotiator
2. Log today's mastery metric
3. Process any captured thoughts

### Weekly Habits
1. Complete the Weekly Review on Friday
2. Update Net Worth on the 1st of each month
3. Process Digital Cleaner bookmarks

### Financial Tracking
1. Log transactions as they happen
2. Review Cash Flow weekly
3. Update Life Calculator monthly

## Troubleshooting

### Hotkeys Not Working
- Due to Chrome extension limitations, POP now has a streamlined set of keyboard shortcuts
- The primary shortcuts are Alt+C (Quick Capture) and Alt+P (Open Dashboard)
- Use the expand/minimize feature and sidebar navigation for quick access to all bubbles
- Press Escape to exit expanded view and return to grid

### Data Not Saving
- Ensure the extension has storage permissions
- Try refreshing the dashboard

### Notifications Not Appearing
- Check Chrome notification settings
- Ensure the extension has notification permissions

### Need Help?
Report issues at the project repository or consult the README.
